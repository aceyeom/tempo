## M1 · 장병내일준비적금 (공짜 돈 앵커) — ⭐ MVP 시드

> `id: M1-savings` · `stat: money` · `difficulty: 1` · `gov_funded: 무료(가입)` · `partner_service_id: bank-cpa`
> 모든 사용자의 **기본(default) 기회**. 난이도 1, 보상 최대 → 첫날 승리로 적합. 적금 디스플레이(→ [00](../00-scope.md) IN #10)와 직결.

- **What:** 정부가 **2024년 이후 납입 원금의 100%를 매칭지원금**으로 더해주는 군 전용 적금. 여기에 은행 기본이자(약 5% 수준, 은행·우대조건별 상이)와 **비과세** 혜택. "공짜로 원금이 두 배" 가 되는 구조라 ROI가 압도적.
- **Eligibility:** 현역병 · 상근예비역 · (구)의무경찰 · 사회복무요원 · 대체복무요원. 1인이 **서로 다른 은행에 최대 2계좌**.
- **Reward (worked math, 시드 — 육군 18개월 가정):**
  | 항목 | 값 |
  |---|---|
  | 월 납입 한도 | 합산 **월 55만원** (은행별 월 최대 30만) |
  | 원금(18개월) | ~990만원 |
  | 정부 매칭(100%) | ~990만원 |
  | 비과세 이자(약 5%) | ~39만원 |
  | **전역 시 예상 수령** | **약 2,015~2,019만원** |
  - 만기(= 전역일)까지 보유해야 매칭 유지. **중도해지 시 매칭지원금·우대 전부 소멸**(원금+기본이자만).
  - 비과세는 **2026-12-31까지 가입** 분에 적용(시드 — 일몰 연장 여부 매년 확인).
- **Schedule:** `frequency: 상시` (입대 후 아무 때나 가입, 빠를수록 매칭 총액 ↑). D-day = **만기일(전역일)**.
- **Cost:** 가입 무료. (납입금은 본인 자산 — "지출"이 아니라 "이전".)
- **Apply:** 은행 앱(KB·IBK·NH·신한·하나·수협·iM뱅크·우체국). **나라사랑카드** 필요. 가입 후 앱에서 납입한도 상향.
- **Referral:** 은행 계좌 개설 **CPA** (→ [06](../06-referral-engine.md)). *정부 돈 경로이므로 윤리 가드 1순위 부합.*

**Quest-path (마일스톤 → 서브퀘스트, atomic·사이즈드):**

1. **개설 (D+0~3)**
   - `S` 나라사랑카드 발급/소지 확인 — 없으면 발급 신청 링크 (10xp)
   - `M` 은행 앱에서 장병내일준비적금 개설 *(service_link: bank-cpa → Tier2 verified)* (25xp)
   - `S` 두 번째 은행 계좌로 합산 55만 채울지 결정 (10xp)
2. **한도 최적화 (D+3~7)**
   - `S` 앱에서 월 납입한도 55만(또는 가능액)으로 상향 (10xp)
   - `S` 자동이체일 = 봉급일 직후로 설정 (10xp)
3. **유지 루프 (매월, 상시 습관 퀘스트로 강등)**
   - `S` 이번 달 납입 정상 입금 확인 (10xp) — 미입금 시 🟡 넛지
4. **만기 추적**
   - `S` 만기(전역) D-day를 위젯에 고정 (10xp)
   - 적금 디스플레이가 매달 예상 잔액 갱신 (→ [01](../01-core-loop.md) §1.4)

- **Source:** korea.kr(납입한도 40→55만 인상), mma.go.kr(병무청), mnd.go.kr, KB장병적금 안내 · **검증일 2026-06-03** · 신뢰도: 공식
- **약점/리스크:** ① **매칭/비과세 조건·한도·일몰은 매년 바뀐다** — 시드값으로 표시하고 `verified_at` 갱신 필수. ② 이자율은 은행·우대조건별 편차가 커 "약 5%"는 근사. ③ 중도해지 페널티를 *명확히* 보여줘야 신뢰(만기까지 유지가 핵심). ④ "2,000만원"은 18개월·만액 가정의 상한치 — 복무기간·납입액에 따라 달라지므로 사용자 입력 기반으로 개인화.

## M2 · 자기개발비 (아무도 다 안 쓰는 예산)

- **What:** 국가가 자기계발 지출의 **80%를 환급, 연 최대 ₩120,000** (본인 부담 20%).
- **Covers:** 도서(잡지/만화 제외) · 시험 응시료(어학·자격증) · 온/오프라인 강좌 수강료 · 학습용품 · 운동용품(최대 6만) · 문화관람.
- **Eligibility:** 육·해·공·해병·국직·카투사·상근 현역 복무자.
- **Apply:** 나라사랑포털 → 자기개발. 제휴사 = 즉시 차감 / 비제휴사 = 영수증 첨부 후 익월 ~15일 e-머니 환급. **연도 이월 안 됨 — 매년 쓰거나 잃는다.**
- **Quest-path:** ① 올해 잔여 한도 확인 ② 쓸 곳 정하기(응시료/강의) ③ 신청 ④ 환급 확인.

## M3 · 전역 목돈 굴리기 (ISA / 청약) — 리터러시 + 웜리드

- **What:** ~2,000만 전역 목돈이 세제 혜택 자리를 갖도록 ISA / 주택청약 계좌 세팅.
- **Reward:** 자산력 XP; 전역 시 증권사 웜리드.
- **Referral:** 증권사 / 은행 CPA.
- **Quest-path:** 리터러시 마이크로 퀘스트 → 계좌 개설.

## M4 · 청년미래적금 (청년도약계좌 후속, 전역 후 매칭 적금)

> `id: M4-youth-future-savings` · `stat: money` · `difficulty: 2` · `gov_funded: true` · `partner_service_id: bank-cpa`
> M1(복무 중 적금)과 **분리된 전역 후 매칭 채널**. 전역 D-90에 해금되는 후속 자산 퀘스트.

- **What:** 2026년 6월 출시된 청년 전용 정부 매칭 적금(청년도약계좌 2025-12-31 종료의 후속). 서민금융진흥원 운영, 시중 ~15개 은행. 월 최대 50만 자유납입 → 정부가 **납입액 6%(일반형)·12%(우대형)** 매칭 + 이자소득 비과세.
- **Eligibility:** 만 19~34세(병역이행 시 복무기간만큼 연장, 최대 ~만 40세). 일반형=개인소득 6,000만 이하 / 우대형=3,600만 이하 등. **대부분 병사는 전역 후 신청**(복무 중 가입 가능 여부는 출시 후 공식 확인 — 시드).
- **Reward (worked, 시드):** 우대형 월 ~6만 매칭 → 3년 기여금 ~216만(일반형 ~108만). 은행금리 6% 가정 시 만기 우대형 ~2,197만 / 일반형 ~2,082만. M1 전역 목돈과 **별도 추가 적립**.
- **Schedule:** 출시 2026-06. 은행 앱 비대면 상시 신청, 연 2회(6·12월) 신규 모집(시드). 만기 3년. 전역 후 첫 모집에 가입이 타이밍 최적.
- **Cost / Apply:** 무료(납입금=본인 자산) · 취급 은행 앱/서민금융진흥원(kinfa.or.kr) · 심사 전산 자동(서류 불필요).
- **Quest-path (D-90 해금):** ① `S` 나이·소득 조건 확인(10xp) → `S` 일반/우대형 기여금 비교(10xp) ② `M` 은행 앱 설치·가입(25xp) *(service_link: bank-cpa)* → `S` 심사 결과 확인(10xp) ③ `S` 월 50만 자동이체 설정(10xp) ④ 월간 `S` 납입 확인(10xp) ⑤ 만기 시 `M` M3(ISA) 연계 굴리기(30xp).
- **Referral:** 은행 계좌 개설 **CPA**(정부 돈 경로 → 윤리 가드 부합, → [06](../06-referral-engine.md) `bank-cpa`).
- **Source:** fsc.go.kr(금융위 출시 보도), imnews.imbc.com(2026-06 보도) · **검증일 2026-06-04** · 신뢰도: 높음(금융위·MBC) / 현역 복무중 가입·중복가입 여부=미검증 시드.
- **약점/리스크:** ① **출시 직후 조건 변동 가능**(비율·한도·소득기준 시행령) → `verified_at` 갱신 필수. ② **복무 중 신청 불가 가능성**(대부분 전역 후) — 미검증. ③ 소득 6,000만 초과 시 일반형도 탈락. ④ 청년도약계좌 보유자 중복 가입 여부 미확인.
