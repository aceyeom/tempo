## E1 · 육군창업경진대회 (FLAGSHIP) — ⭐ MVP 시드

> `id: E1-army-startup` · `stat: edge` · `difficulty: 5` · `gov_funded: 무료` · `partner_service_id: null`
> 가장 깊은 표현의 hook: **"네 아이디어로 휴가 따자."** 난이도 5 → 생성기 **에스컬레이션 모델**로 최초 계획(→ [03](../03-opportunity-engine.md) §3.4). 마감이 빡센 단일 `deadline` 형이라 페이싱이 핵심.

- **What:** 육군이 주최하는 창업 경진대회. 국가 *도전! K-스타트업* 파이프라인의 관문이며 진짜 엑싯 사례 존재(예: 링티 창업자가 과거 수상). 2026년 **제15회**.
- **Eligibility:** 육군(장교·부사관·병). 팀 단위 참가(개인 단독 불가가 일반적 — 라이브 요강 확인). **개인당 최대 3건 신청 가능하나 중복 수상 불가**(2026 요강 기준).
- **Reward:** 본선 수상자 → **포상휴가 보통 2~5일(용사, 부대 내규·부대별 상이)** + 상금(대상 ~500만, 총상금 ~2,500만 규모, 시드) + 창업멘토링캠프(공가) + 창업지원기관 연계 → 국방 Start-up 챌린지(하반기) → 도전! K-스타트업.
- **Schedule (2026 제15회, 검증됨):**
  | 단계 | 일정 |
  |---|---|
  | 서류 접수 | 2026-02-09 ~ 04-10 14:00 |
  | 서류 발표 | 2026-04-23 18:00 (홈페이지 공지) |
  | 본선(온라인 평가) | 2026-05-12 ~ 05-15 중 1일 |
  | 시상식 | 2026-05-27 |
  - `frequency: 반기` — 육군은 **하반기 사이클**도 운영. (봄 사이클은 종료 → 다음 사이클로 리타겟이 기본 재계획)
- **Cost:** 무료.
- **Apply:** **army-startup.co.kr** '접수하기' 탭 (예선: 폼 + **5분 유튜브 영상**, 유튜브 외 플랫폼 불가 → 본선: 발표영상 + 온라인 Q&A).
- **Referral:** 없음 (정부 주최, 비-리퍼럴).

**Quest-path (마일스톤 → 서브퀘스트, 마감 역산 페이싱):**

1. **아이디어 (접수 4주 전)**
   - `S` 아이디어 한 줄로 적기 (15xp, edge ×1.5)
   - `M` 비슷한 서비스/시장 30분 조사 (38xp)
   - `S` 팀 필요 인원·역할 정하기 (15xp)
2. **팀 빌딩 (3주 전)**
   - `M` 팀원 1~4명 컨택 (38xp, people)
   - `S` 역할 분담 1줄 합의 (15xp)
3. **사업모델 (2주 전)**
   - `L` 사업모델 1장(문제·해결·시장·수익) 작성 (90xp)
   - `M` 피드백 1명에게 받기 (38xp)
4. **제출물 (1주 전)**
   - `L` 5분 발표 대본 작성 (90xp)
   - `L` 발표 영상 촬영 (90xp)
   - `M` 유튜브 업로드 + 접수 폼 제출 *(마감 D-day 하드 클램프)* (38xp)
5. **본선 (서류 합격 시 트리거되는 에스컬레이션 재계획)**
   - `M` 예상 Q&A 10개 작성 (38xp)
   - `L` 온라인 Q&A 리허설 (90xp)

- **Source:** army-startup.co.kr(2026 제15회 요강) · **검증일 2026-06-03** · 신뢰도: 공식 / 휴가·상금 일부는 커뮤니티·과거 회차 기반(시드)
- **약점/리스크:** ① **팀 필수** → 솔로 병사에겐 진입장벽; 앱이 팀 빌딩을 도울 수 없으면 경로가 막힌다(부대 내 모집 한계). ② 마감 단일형이라 **사이클을 놓치면 다음 반기까지 대기** — 재계획은 "다음 회차 D-day"로 리타겟해야지 거절하면 안 됨(→ [03](../03-opportunity-engine.md) §3.0-2). ③ 상금·휴가 수치는 회차별 변동 — 시드. ④ 난이도 5라 완료율이 낮을 것 → 담력 XP를 후하게(스크린샷 트로피).

> **관련 (Phase 2 확장):** 동일 포맷의 **해군 창업경진대회**(navy-startup.co.kr), **공군 창업경진대회**(airforce-startup.co.kr)가 존재 → 군별 분기 항목으로 추가 예정(→ [appendix](../appendix-sources.md)).
